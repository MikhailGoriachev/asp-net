Базовый класс таблицы

The .table class adds basic styling (light padding and horizontal dividers) to a table:

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

Таблица с "чередующимися" строками

Класс .table-striped добавляет "чередование" к стилям оформления таблицы:

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

Таблица с границами

Класс .table-bordered добавляет границы к строкам и ячейкам таблицы:

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

Строки с реакцией на наведение курсора мыши (hover)

Класс .table-hover добавляет реакцию строки на наведение мыши на строку (серый фон, когда курсор мыши проходит над строкой):

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

Несколько классов, примененных к таблице

Firstname Lastname Email
John Doe john@example.com

Темная таблица

Класс .table-dark создает таблицу с черным фоном:

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

Темная таблица с чередованием и ховером

Классы .table-striped и .table-hover добавляют эффекты "чередования" и выделения строки:

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com
Tim Yanson tim@example.com

Таблица без границ

Класс .table-borderless убирает границы строк и ячеек таблицы:

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

Контекстые классы для цвета текста

Контекстные классы м.б. использованы для строк или ячеек. Классы: .table-primary, .table-success, .table-info, .table-warning, .table-danger, .table-active, .table-secondary, .table-light and .table-dark:

Firstname Lastname Email
Default Defaultson def@somemail.com
Primary Joe joe@example.com
Success Doe john@example.com
Danger Moe mary@example.com
Info Dooley july@example.com
Warning Refs bo@example.com
Active Activeson act@example.com
Secondary Secondson sec@example.com
Light Angie angie@example.com
Dark Bo bo@example.com

Компактная таблица

Класс .table-sm class делает таблицу компактнее, уменьшая внутренние отступы на половину:

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

Адаптивная таблица

Класс .table-responsive при необходимости добавляет таблице полосу прокрутки:

# Firstname Lastname Age City Country Sex Example Example Example Example Example Example Example Example Example Example Example Example

3 Anna Pitt 35 New York USA Female Yes Yes Yes Yes Yes Yes Yes Yes Yes Yes Yes Yes

Класс .table-responsive-sm создает адаптивную таблицу с прокруткой, добавляемой при ширине окна браузера меньше 576px
